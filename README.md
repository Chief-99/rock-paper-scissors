# Rock, Paper, Scissors Game

A simple rock, paper, scissors game built with Javascript and HTML. This project helped with learning the basics of JavaScript, working with user inputs and adding interactivity to web pages. 

## Features
- Play a game of rock, paper, scissors against the computer.
- Randomised computer choices for fair gameplay.
- Case-insensetive user input (e.g., 'ROCK', 'rock', or 'rock' all work).
- Displays game results: Win, lose or tie.
